# MyCask Brand Guide

## Color Palettes

### ✅ Light Mode Palette

- Primary Background: `#FFFFFF` (White)
- Primary Accent (Deep Green): #0F3B2E
- Secondary Accent (Luxury Gold): #C49A3B
- Dark Gold for Depth: #8A6A2A
- Whiskey Amber (Highlight): #A86C2D
- Text (Primary): #1A1A1A (Near Black)
- Text (Secondary): #4F4F4F (Gray for muted text)
- UI Highlights (Buttons, Links): #E2C76E (Soft Gold)

### ✅ Dark Mode Palette

- Primary Background: #0F3B2E (Deep Green)
- Secondary Background: #1E4D3B (Slightly lighter green for panels)
- Primary Accent (Gold): #C49A3B
- Highlight Gold: #E2C76E
- Whiskey Amber: #A86C2D
- Text (Primary): #FFFFFF
- Text (Muted): #CFCFCF
- Borders/Dividers: #2A5A47

### ✅ Typography Suggestions

- Primary Font: Modern serif or clean sans-serif (e.g., Playfair Display or Inter)
- Accent Font: Use serif for headings to keep classy feel.
